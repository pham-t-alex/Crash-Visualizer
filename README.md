# Crash-Visualizer
 
Frontend Setup

`cd frontend`

`npm install`

Backend Setup

`cd backend`

`python3 -m venv venv`

`.\venv\Scripts\Activate` (Windows) /
`source venv/bin/activate` (Linux/Mac)

`pip install -r requirements.txt`

Frontend Start

`cd frontend`

`npm run dev`

Backend Start

`cd backend`

`.\venv\Scripts\Activate` (Windows) /
`source venv/bin/activate` (Linux/Mac)

`python app.py`
